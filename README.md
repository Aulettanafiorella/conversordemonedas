# Conversor de Monedas

## Descripción
Aplicación web que permite convertir monedas de forma dinámica mediante lógica de cálculo de tipo de cambio. Cuenta con una interfaz simple y amigable para el usuario.

## Tecnologías utilizadas
- JavaScript
- HTML5
- CSS3

## Funcionalidades
- Conversión de monedas en tiempo real
- Cálculo dinámico de tipo de cambio
- Validación de datos ingresados
- Interfaz clara y fácil de usar

## Objetivo del proyecto
Demostrar implementación de lógica en frontend, manejo de datos y diseño de interacción con el usuario.
